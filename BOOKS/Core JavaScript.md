## 01 데이터 타입의 종류
### Remember
- data type
  - Primitive type
    - Number / String / Boolean / null / undefined / Symbol
    - 할당이나 연산 시 복제
    - 값이 담긴 주소값 바로 복제
    - immutability
  - Reference type - Object
    - Array / Function / Date / RegExp / Map, WeakMap / Set, WeakSet
    - 참조
    - 값이 담긴 주소값들로 이루어진 묶음을 가리키는 주소값 복제
